## 摘要
域泛化从多个源域中学习，以泛化至未见过的目标域。然而，它常忽略源域与目标域标签不匹配的现实情况。于是，开放集域泛化被提出，以识别未见域中的未知类别。一种简单的方法是训练一对多分类器来分离每个类别，并将离群值检测为未知。然而，少量正样本与大量负样本之间的不平衡使决策边界偏斜向正样本，导致模型过度拒绝分布外数据，甚至包括未见域中的已知类别。本文提出了一种新颖的元学习策略，称为联合域类别匹配的双元元学习（MEDIC），该策略同时考虑域间和类间任务分割的隐式梯度匹配，以找到对域和类别均平衡的最优边界。实验结果表明，MEDIC不仅在开放集场景中优于先前方法，而且保持了有竞争力的闭集泛化能力。

## Abstract
Domain generalization learns from multiple source domains to generalize to unseen target domains. However, it often neglects the realistic case of label mismatch between source and target. Open set domain generalization is then proposed to recognize unseen classes in unseen domains. A simple approach trains one-vs-all classifiers to separate each class and detect outliers as unknown. Yet, the imbalance between few positive samples and many negative samples skews the decision boundary towards the positive ones, leading the model to over-reject out-of-distribution data, even from known classes in unseen domains. In this paper, we propose a novel meta-learning stategy called dualistic MEta-learning with joint DomaIn-Class matching (MEDIC), which considers implicit gradient matching towards inter-domain and inter-class task splits simultaneously to find optimal boundaries balanced for both domains and classes. Experimental results show that MEDIC not only outperforms prior methods in open set scenarios, but also maintains competitive close set generalization ability.

### 1. 论文核心问题与研究背景

- **核心问题**: 论文聚焦于**开放集域泛化 (Open Set Domain Generalization, OSDG)** 这一现实且具有挑战性的场景。传统域泛化 (DG) 假设源域和目标域的标签空间一致 (闭集)，而 OSDG 要求模型在泛化到未见目标域时，不仅能正确分类已知类别，还能识别出训练时未出现过的未知类别。
- **研究背景与动机**: 现有 OSDG 方法常用一对多 (One-vs-All) 分类器来分离各类别并检测未知类。然而，论文指出这种方法存在严重缺陷：每个二分类器中，正样本（属于该类）极少而负样本（属于其他所有类）极多，这种**极端的数据不平衡会导致决策边界严重偏向正样本**。其后果是，在未见域中，模型极易将属于已知类别但分布稍有变化的样本过度拒绝为“未知”，从而严重损害分类性能。因此，本文旨在解决此不平衡问题，学习一个对域和类别都平衡的决策边界。

### 2. 方法论与关键技术

- **核心思想**: 提出了一种名为 **MEDIC** 的双元元学习策略，其核心是通过**隐式的梯度匹配 (Implicit Gradient Matching)**，同时考虑域间 (Inter-domain) 和类间 (Inter-class) 的任务分割，以找到对域和类别双重平衡的最优参数点。
- **关键技术细节**:
    - **任务构造**: 传统元学习（如 MLDG）只按域划分任务（meta-train, meta-test），MEDIC 在此基础上进一步按类别划分。它将源域数据切分为两部分：`D1`, `D2`，每部分再按类别切分为不相交的 `D1_a`, `D1_b` 和 `D2_a`, `D2_b`。任务被重组为 `(D1_a, D2_b)` 对和 `(D2_a, D1_b)` 对，确保每一步梯度的计算都同时涉及来自不同域和不同类的样本。
    - **双元梯度匹配**: 元优化的目标被推导为两项之和：一项是各任务预期损失的最小化，另一项是不同任务梯度之间点积的最大化（即正则化项）。这个正则化项实现了梯度匹配：`F1'·F2'` 实现了**域内类间匹配**，`F1'·G1'`等实现了**类内域间匹配**。通过同时最大化这类点积，模型被驱动到参数空间的一个“共识”区域，在该区域中，优化任一任务（无论是域迁移还是类变化）不会损害其他任务。
    - **MEDIC++ 框架**: 将 MEDIC 推广为更灵活的 **MEDIC++**。它允许在一个 inner loop 中执行多个步骤（step），每步包含多个任务（task）。论文通过理论分析与实验证明，适量增加步数（如3步）能显著增加任务间的梯度匹配对数量，但当步数过多时收益递减，且可能导致不同类特征趋同，反而有害。
    - **自适应任务采样**: 为了更有效地处理易混淆类，提出基于类别转移概率矩阵的采样策略。该矩阵记录了模型将 `i` 类错分为 `j` 类的概率，采样时会优先将易混淆的类别对分配到不同的步骤中，从而增加它们之间梯度匹配的频率。
    - **损失函数**: 训练使用联合损失 `L_all = L_ce + L_ova`。其中 `L_ce` 是标准交叉熵损失用于闭集分类，`L_ova` 是专门设计的 Open Set 损失，它在一对多分类器上优化，确保样本在其正确类通道上得分高，并明确将最难区分的负类通道得分压至最低。
