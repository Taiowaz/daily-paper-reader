## 摘要
领域泛化从多个源域中学习以泛化到未见过的目标域。然而，它常常忽略了源域和目标域之间标签不匹配的现实情况。于是，开放集领域泛化被提出，用于识别未见域中的未见类别。一种简单的方法训练一对多分类器来分离每个类别，并将离群点检测为未知类。但是，少数正样本与大量负样本之间的不平衡使得决策边界偏向正样本，导致模型过度拒绝分布外数据，甚至包括未见域中的已知类别数据。在本文中，我们提出了一种新颖的元学习策略，称为双重元学习与联合域-类别匹配（MEDIC），它同时考虑域间和类别间任务分割的隐式梯度匹配，以找到对域和类别都平衡的最优边界。实验结果表明，MEDIC不仅在开放集场景中优于先前的方法，而且还保持了有竞争力的闭集泛化能力。

## Abstract
Domain generalization learns from multiple source domains to generalize to unseen target domains. However, it often neglects the realistic case of label mismatch between source and target. Open set domain generalization is then proposed to recognize unseen classes in unseen domains. A simple approach trains one-vs-all classifiers to separate each class and detect outliers as unknown. Yet, the imbalance between few positive samples and many negative samples skews the decision boundary towards the positive ones, leading the model to over-reject out-of-distribution data, even from known classes in unseen domains. In this paper, we propose a novel meta-learning stategy called dualistic MEta-learning with joint DomaIn-Class matching (MEDIC), which considers implicit gradient matching towards inter-domain and inter-class task splits simultaneously to find optimal boundaries balanced for both domains and classes. Experimental results show that MEDIC not only outperforms prior methods in open set scenarios, but also maintains competitive close set generalization ability.

### 1. 论文的核心问题与整体含义（研究动机和背景）

本文聚焦于**开放集领域泛化（Open Set Domain Generalization, OSDG）**这一更具挑战性的实际任务。传统的领域泛化（DG）通常假设训练域（源域）和测试域（目标域）的标签空间一致（闭集），但现实中，部署后的模型很可能遇到训练时从未见过的新类别。这就同时带来了**数据分布偏移（域漂移）**和**标签空间偏移（类别漂移）**两大问题。

现有方法常采用一对多（one-vs-all）分类器来识别未知类，但正负样本的严重不均衡会使决策边界过度偏向少数正样本。这会导致对于未见域中的已知类样本，模型也会因呈现“分布外”特征而将其**过度拒绝**为未知类，严重损害闭集分类的准确性。因此，论文的核心动机是：**如何学习一个对域和类别双重鲁棒的决策边界，使模型能在准确识别已知类的同时有效检测未知类。**

### 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

论文提出了一个名为 **MEDIC（Dualistic Meta-learning with joint Domain-Class matching，双重元学习与联合域-类别匹配）** 的元学习框架。

#### 核心思想
该方法的核心创新在于**同时执行域间（inter-domain）和类间（inter-class）的隐式梯度匹配**。其基本思路是：通过元学习，寻找一个参数空间区域，使得从不同域和不同类划分出的不同任务之间的梯度方向对齐（夹角小），这样对一个任务的优化不会损害其他任务，从而实现“共赢”。MEDIC 将源数据按域和类两个维度交叉划分成多个子任务，并通过精心设计的元学习内/外循环来对齐这些任务的梯度。

#### 关键技术细节与算法流程
1.  **任务划分与重组（Task Splitting）**：
    给定多个源域数据，MEDIC首先将数据按**域**划分，然后再按**类别**进行二次划分，形成更精细的任务。例如，将来自域A和域B的数据，各自分成包含不同类别的两部分，最终得到四个任务 (如 A1, A2, B1, B2)。其重组策略要求元训练集包含来自不同域和不同类的任务组合（如 A1, B2），而元测试集则包含互补的任务组合（如 A2, B1），以此强制模型同时学习跨域和跨类的泛化能力。

2.  **双重梯度匹配（Dualistic Gradient Matching）机制**：
    MEDIC 采用一种类似 Reptile 的元学习算法，但深化为**多步（multi-step）内循环**。一个训练迭代的流程如下：
    *   **内循环（Inner Loop）**：将总任务分成 *n* 个步骤（step），每个步骤包含若干个子任务。模型在当前参数 \theta 上，依次用每个步骤的任务损失进行梯度下降，得到更新后的参数 \hat{\theta}。
    *   **外循环（Outer Loop）**：原始参数 \theta 向 \hat{\theta} 的方向进行更新，即 \theta \leftarrow \theta + \epsilon(\hat{\theta} - \theta)。
    论文通过数学推导证明，这种多步更新机制等价于最大化不同步骤之间梯度的点积（即鼓励梯度方向一致），从而实现了**步骤间（inter-step）**的梯度匹配，并最终扩展为所有**任务间（inter-task）**的梯度匹配。研究还发现，内循环步数n的增加与梯度匹配任务对数呈正相关，但边际收益递减，因此只需少量步骤即可达到良好效果。

3.  **自适应任务采样（Adaptive Task Sampling）**：
    为了更有效地匹配易混淆类别之间的梯度，MEDIC引入了一种自适应采样策略。它首先构建一个类别间的非对称转移概率矩阵（某个类被误判为另一类的平均概率），然后在划分任务时，倾向于将转移概率高的易混淆类别分配到不同的步骤中，从而最大化它们之间的梯度匹配频率。

4.  **开放集损失函数与推理**：
    *   **损失函数**：模型由一个闭集分类器（使用交叉熵损失 L_ce）和一个多二值分类器（使用一对多损失 L_ova）组成。总损失为 L_all = L_ce + L_。多二值分类器为每个已知类训练一个独立的二分类器，用于判断样本是否属于该类。
    *   **推理（Inference）**：测试时，样本首先通过闭集分类器获得其最可能的已知类。然后，根据该类的二值分类器的置信度得分来判断：如果得分高于阈值，则分类为该已知类；否则，判定为未知类。
