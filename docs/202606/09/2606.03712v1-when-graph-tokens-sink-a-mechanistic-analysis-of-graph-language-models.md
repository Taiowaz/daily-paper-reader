## 摘要
图语言模型（Graph Language Models, GLMs）已成为将大语言模型（Large Language Models, LLMs）适配到图学习任务的一个有前景的方向。通过将图拓扑和节点信息转化为图token，GLMs使LLMs能够联合处理结构化的图输入和文本指令。然而，LLMs如何在内部解释这些图token，以及图token是否充当图结构的有意义载体，目前仍不清楚。在这项工作中，我们通过代表性GLMs架构中的图token行为，分析了LLMs如何处理图信息。

研究发现。我们发现，GLMs中图token的内部显著性并不等同于图信息的利用。图汇token（graph sink tokens）始终表现为激活层面的异常值：它们可以通过沿少量隐藏状态维度的大幅激活值来识别，并且偏向于早期的图token位置。然而，这种激活层面的显著性并不意味着这些token是图信息的主要载体。与语言和视觉-语言模型中的经典注意力汇不同，图汇token不一定会从查询token吸引最大的注意力权重。通过剪枝、重定位和交换干预，我们表明，对于下游预测，图汇token并非最重要的语义或结构token。

意义。综合来看，这些结果表明，在当前GLMs将图结构映射到LLM的token空间后，所得到的图token表示并未自然形成完全可用的拓扑感知的内部表示；相反，它们表现出激活层面显著性与图语义效用之间的脱耦。这种脱耦揭示了现有图token构建、放置和对齐机制中的局限性。

## Abstract
Graph Language Models (GLMs) have become a promising direction for adapting Large Language Models (LLMs) to graph learning tasks. By transforming graph topology and node information into graph tokens, GLMs allow LLMs to jointly process structured graph inputs and textual instructions. Yet, it remains unclear how LLMs internally interpret these graph tokens and whether graph tokens act as meaningful carriers of graph structure. In this work, we analyze how LLMs process graph information through graph-token behavior in representative GLM architectures.   Findings. We find that the internal saliency of graph tokens in GLMs is not equivalent to graph information utilization. Graph sink tokens consistently emerge as activation-level outliers: they can be identified by massive activation values along a small set of hidden-state dimensions and are biased toward early graph-token positions. However, this activation-level saliency does not imply that these tokens are the main carriers of graph information. Unlike classical attention sinks in language and vision-language models, graph sink tokens do not necessarily attract the largest attention weights from query tokens. Through pruning, repositioning, and swapping interventions, we show that graph sink tokens are not the most important semantic or structural tokens for downstream prediction.   Implications. Together, these results suggest that after current GLMs map graph structure into the LLM token space, the resulting graph-token representations do not naturally form a fully usable topology-aware internal representation; instead, they exhibit a decoupling between activation-level saliency and graph-semantic utility. This decoupling points to limitations in existing graph-token construction, placement, and alignment mechanisms.

- **核心问题**：当前 GLM 假设非欧几里得的图结构能被“压平”为顺序 token 并忠实解析，但 LLM 的 Transformer 机制容易产生非语义的注意力汇（attention sink）。本文追问：**当图 token 在 LLM 内部变得“异常高激活”时，模型是否真的在利用它进行拓扑推理，还是仅仅在构建架构层面的伪影？**

- **关键技术细节**：
  - **图汇 token 定义**：根据隐藏状态的激活幅度定义。给定图 token 索引集合 \( I_g \)，第 \( l \) 层隐藏状态 \( x_{l-1}^j \)，通过预设的 sink 维度集 \( D_{\text{sink}} \) 和阈值 \( \tau \)，若 token 在某个 sink 维度上的 RMSNorm 归一化值超过阈值，则判定为图汇 token（公式 1、2）。本文中 \( \tau = 15.0 \)。
  - **分析的两个代表性 GLM**：
    - **LLaGA（节点对齐式图 token）**：将中心节点及其 k‑hop 邻居采样为固定长度的树状序列，不足时用 [PAD] 填充。
    - **TEA‑GLM（编码器生成式图 token）**：先用 GNN 聚合邻居信息，再线性投影为固定数量的可学习图 token 嵌入，并通过对比学习与 LLM 空间对齐。
  - **分析手段**：
    - 计算 sink token 的激活值分布、位置分布。
    - 可视化 query‑to‑graph 注意力图及层级注意力模式。

    - 进行 Logit Lens 分析，将图 token 的隐藏状态投射到词汇表，检查其解码出的语义内容。
    - 在后剪除场景下重新检测 sink token 的重新分布。

### 6. 论文的主要结论与发现
- **图汇 token 普遍存在且表现为激活异常值**：在 LLaGA 和 TEA‑GLM 中，少部分隐藏维度（如维度 1512、2533）上出现剧烈的激活尖峰，这些 token 偏向位于图 token 序列的前端（尤其是 [PAD] token 或前几个编码器生成的 token）。
- **激活显著性不等于注意力主导地位**：高激活的图汇 token 并未稳定获得来自查询 token 的最大注意力权重，尤其是 TEA‑GLM 中注意力更倾向于较后位置的图 token。
- **图汇 token 不是主要的图语义/结构信息载体**：
  - 剪除 Top‑2 sink token 几乎不影响下游性能，而随机剪除非 sink token 常导致更大的性能下降（尤其在 LLaGA 上）。
  - Swap 和重定位干预对性能影响微弱。
  - 随着图序列中真实邻居信息增多（非填充 token 比例升高），分配给 sink token 的注意力下降，进一步说明其与信息内容的无关性。
- **Logit Lens 揭示 token 语义贫乏**：TEA‑GLM 的图 token 隐藏状态大多解码为通用词汇（如 “paper”）或碎片子词，而非任务相关的类别或图结构概念。
- **后剪除现象**：LLaGA 在剪除原始 sink 后，激活异常会重新分散到多个位置；而 TEA‑GLM 则几乎不再产生 sink 级激活，说明两种模型的 sink 形成机制存在差异。
- **总体含义**：以上发现共同指向一个核心结论：当前 GLM 在将图映射到 token 序列后，并未自动形成拓扑感知、可直接利用的内部表示，激活层面的“繁荣”与图语义效用之间存在严重脱耦。

### 7. 优点
- **机制研究视角新颖**：首次系统定义了图领域中的“图汇 token”，并将 LLM 中已知的注意力汇现象与 GLM 的图信息传递问题联系起来。
- **多层次、多手段的交叉分析**：结合激活幅度、注意力可视化、因果干预和语义解码，证据链坚实。
- **实验设计公平且自对照**：不依赖于与其它模型的横向比较，而是通过同一模型的干预前后对比得出因果结论，减少了混杂因素。
- **覆盖代表性架构**：同时分析节点对齐式和编码器生成式两类主流 GLM，增强结论的普适性警示。
- **清晰的发现与区分**：明确区分了激活显著性、注意力主导地位和语义效用三个概念，为后续 GLM 设计提供了可量化的诊断维度。

### 8. 不足与局限
- **模型覆盖有限**：仅详细分析了 LLaMA 家族上的两种 GLM，未涉及基于其他 LLM 骨干或更复杂图编码方式的模型，结论的通用性仍需进一步验证。
- **任务与数据集范围**：局限于引文网络的节点分类和链接预测，未覆盖图分类、知识图谱问答等任务，以及非引文领域（如分子、社交网络）的数据。
- **TEA‑GLM 表现较弱**：该模型在部分数据集上的零样本节点分类准确率极低（如 Cora 仅 13.33%），可能影响对其 sink 行为分析的效度，其在强任务场景下的行为未知。
- **干预程度与现实性的权衡**：剪除操作会改变序列长度，虽设有非 sink 剪除作为对照，但该对照并未完全隔离激活幅值本身的影响；重定位也会破坏模板的拓扑顺序，但其影响可能被模型的不敏感性所掩盖。
- **缺少正向解决方案**：论文重在诊断问题，未提出改进图 token 构建、放置或对齐的具体方法，应用层面价值停留在警示与指引。
- **Logit Lens 的解读主观性**：解码出的词元语义解释可能存在一定主观成分，且概率普遍较低，是否足以支撑“语义贫乏”的强论断值得商榷。
