## 摘要
文本属性图（TAGs）广泛应用于许多现实世界领域，在TAGs上的学习需要联合建模文本语义和图结构。建模TAGs的标准方法是结合语言模型（LM）和图神经网络（GNN），但联合训练计算成本高且难以扩展。数据集蒸馏是降低训练成本的一种有前景的方法，但现有方法并不适合TAGs，因为它们通常为单一模态设计，或在蒸馏过程中仍需在完整数据集上反复训练昂贵的LM-GNN模型。为了解决这个问题，我们提出了TaLK，一种有效的TAGs数据集蒸馏方法，它将LM与图感知神经正切核耦合。这种设计实现了高效的数据集蒸馏，避免了在完整数据集上的重复联合训练，同时反映文本和结构信息以实现有效的TAG学习。在多个TAG基准上的实验表明，TaLK始终优于现有基线，仅使用1%的合成数据即可达到高达97%的完整数据集性能。

## Abstract
Text-attributed graphs (TAGs) are widely used in many real-world domains, and learning on TAGs requires jointly modeling text semantics and graph structure. A standard approach for modeling TAGs is to combine a language model (LM) and a graph neural network (GNN), but joint training is computationally expensive and difficult to scale. Dataset distillation is a promising way to reduce training costs, but existing methods are not well suited to TAGs because they are typically designed for a single modality or still require repeatedly training expensive LM-GNN models on the full dataset during distillation. To address this, we propose TaLK, an effective dataset distillation method for TAGs that couples an LM with a graph-aware neural tangent kernel.This design enables efficient dataset distillation, avoiding repeated joint training on the full dataset while reflecting both textual and structural information for effective TAG learning.Experiments on multiple TAG benchmarks show that TaLK consistently outperforms existing baselines and achieves up to 97% of full-dataset performance with only 1% synthetic data.

### 1. 论文的核心问题与整体含义

本论文聚焦于**文本属性图（Text-Attributed Graphs, TAGs）的数据集蒸馏问题**。TAGs广泛存在于学术网络、社交媒体和电商系统中，其节点包含丰富的文本，边代表节点间关系。由于对TAGs进行学习需要联合建模文本语义和图结构，目前主流方法是将预训练语言模型（LM）与图神经网络（GNN）进行端到端联合训练。然而，这种联合训练面临严峻的扩展性挑战：
-   **计算与内存开销大**：更新LM参数时，需为邻域节点保留大量中间状态，在处理高连通度的大规模图时尤为严重。
-   **现有数据集蒸馏方法不适配**：数据集蒸馏是一种通过合成少量、精炼的训练样本来降低模型训练成本的技术，但现有方法要么为单一模态（纯文本或纯图）设计，要么在蒸馏过程中仍需在完整数据集上反复训练昂贵的LM–GNN模型，从而继承了上述扩展性瓶颈。

因此，**本研究的核心目标是设计一种高效、适用于TAGs的数据集蒸馏方法**，使其能保持端到端联合训练的优势，同时避免在完整数据集上的重复训练，从而大幅降低训练成本。

### 2. 论文提出的方法论

TaLK（Text-attributed graph dataset distillation via coupling LM with graph-aware Kernel）的核心思想是：**在双层优化框架中，将语言模型与图感知的神经正切核（Graph-Aware Neural Tangent Kernel）耦合**，在保证蒸馏有效性的同时，规避外层循环中昂贵的GNN训练。

**关键技术细节与算法流程如下：**

-   **蒸馏框架**：采用双层优化。
    -   **内层循环**：在合成的TAG `S`（特征为可学习的令牌级嵌入 `X'`，图结构固定为单位矩阵 `I`）上，联合训练一个LM编码器 `f_ϕ` 和一个GNN `g_ψ`。这一步确保了LM参数的更新与标准的联合训练流程一致。
    -   **外层循环**：优化合成数据集 `S`，目标是使在 `S` 上训练的模型在全量数据集 `T` 上表现良好。为避免在 `T` 上重复训练，TaLK在此环节替换了GNN。

-   **图感知的核岭回归目标**：
    1.  利用内层循环训练好的LM编码器 `f_ϕ`，获取全量数据集和合成数据集的文本表征 `H_T` 和 `H_S`。
    2.  应用**结构基神经正切核**（SNTK）`K(·, ·)`，该核函数能将图结构信息（通过邻域聚合）注入到核矩阵计算中，但无需训练GNN。基于此计算合成-合成核矩阵 `K_{SS}` 和全量-合成交叉核矩阵 `K_{TS}`。
    3.  利用核岭回归（KRR）的闭式解，通过 `K_{TS}` 和 `K_{SS}` 对全量数据集 `T` 的标签进行预测：`Ŷ = K_{TS} (K_{SS} + λI)^{-1} Y'`。外层优化目标 `L_{KRR}` 是最小化该预测在真实标签上的误差。这种方式使得合成数据能同时编码文本语义和图结构信息。

-   **批量梯度注入**：
    -   **问题**：外层核目标依赖全局合成集上下文，普通的小批量训练会改变优化问题本身。
    -   **解决方案**：首先基于所有合成样本的全局表征计算总损失 `L_{KRR}` 和全局表征 `H_S` 上的梯度 `G`。然后，将全局梯度 `G` 按批次切片 `G^{(b)}`，并分别注入作为每个合成批次 `X'^{(b)}` 经过LM反向传播时的上游梯度。这样既保留了全局优化目标，又实现了可按小批量进行的LM反向传播，节约了内存。
