## 摘要
图神经网络（GNN）将其输入输出参数紧密耦合到数据集特定的特征空间和目标集，在不同数据集上表现出有限的可迁移性。相比之下，语言模型（LM）通过统一的输入输出接口灵活泛化，这促使近期尝试将语言模型适配到图任务上。然而，现有方法难以编码整个图的信息，导致潜在的信息损失和次优的图理解。在本文中，我们提出了一种新颖的权重级信息注入范式，用于将语言模型适配到图任务。该范式通过生成与隐藏表示直接交互的任务特定权重更新来注入全图信息。遵循低秩适配（LoRA）实例化该范式，我们引入了GaRA，一个图感知的LoRA生成模型。GaRA以原始图结构为条件构建低秩权重更新，并约束生成更新的范数，从而注入全图信息并避免权重生成中的优化偏差。实证研究表明，GaRA在零样本图学习任务上持续优于基线方法。

## Abstract
Graph neural networks (GNNs) tightly couple their input-output parameters to dataset-specific feature spaces and target sets, exhibiting limited transferability across different datasets. In contrast, language models (LMs) generalize flexibly via a unified input-output interface, motivating recent attempts to adapt LMs to graph tasks. However, existing methods struggle to encode whole-graph information, leading to potential information loss and suboptimal graph understanding. In this work, we propose a novel weight-level information injection paradigm for adapting LMs to graph tasks. This paradigm injects whole-graph information by generating task-specific weight updates that interact directly with hidden representations. Instantiating this paradigm following low-rank adaptation (LoRA), we introduce GaRA, a Graph-aware LoRA generation model. GaRA constructs low-rank weight updates conditioned on the original graph structures and constrains the norm of the generated updates, thus injecting whole-graph information and avoiding the optimization bias in the weight generation. Empirical studies demonstrate that GaRA consistently outperforms baselines on zero-shot graph learning tasks.

### 2. 方法论：权重级信息注入与 GaRA

**核心思想**  
- 提出**权重级注入范式**，通过以全图特征生成权重更新矩阵，直接作用于 LM 的隐藏表示，弥补输入级注入丢失的全局信息。  
- 实例化为**图感知 LoRA 生成模型（GaRA）**，遵循低秩适配（LoRA）的效率优势。

**关键技术细节**  

1. **整体架构**（见图 2）  
   - 基于 UniGTE 输入注入框架，额外添加图感知权重生成模块。  
   - 流程：  
     - 对输入图用 GNN 提取节点特征，最大池化得到图级表示 $h$。  
     - 通过非线性映射 $g$ 生成低秩矩阵 $\mathbf{G}$。  
     - $\mathbf{G}$ 与 LoRA 学习矩阵 $\mathbf{L}$ 结合，形成最终权重更新 $\Delta\mathbf{W} = (\mathbf{L} + \mathbf{G})\mathbf{R}^\top$（选择左矩阵生成，右矩阵固定）。  
     - 权重更新作用于 LLM 编码器的特定层（实验中第 21 层效果最佳）。

2. **范数重标定（Norm Rescaling）**  
   - **问题**：单纯权重生成存在方向保持的范数放大（方向保持更新）偏差，即优化过程可通过扩大生成矩阵范数降低训练损失，导致原始权重被稀释，模型过拟合。  
   - **理论证明**（定理 4.1 及推论 4.2）：存在参数更新方向使得 $\Delta\mathbf{W} \gets (1+\epsilon)\Delta\mathbf{W}$，而 LoRA 自身的权重衰减无法有效约束。  
   - **解决方案**：将生成矩阵 $\mathbf{G}$ 的平均列范数固定为 LoRA 左矩阵 $\mathbf{L}$ 的平均列范数：  
     $\mathbf{M} = \frac{\text{avg}_j(\|\mathbf{L}_{\cdot,j}\|)}{\text{avg}_j(\|\mathbf{G}_{\cdot,j}\|)}\mathbf{1}^\top$，  
     最终 $\Delta\mathbf{W} = (\mathbf{L} + \mathbf{G} \odot \mathbf{M})\mathbf{R}^\top$。  
   - 此设计无需额外超参数，自然以 $\mathbf{L}$ 的范数作为基准尺度。

3. **训练策略**  
   - 采用指令微调，最大化生成标签序列的对数似然（交叉熵损失）。  
   - 可训练参数包括 LoRA 低秩矩阵和权重生成器参数。

**公式关键点**（文字说明）  
- 图表示提取：$\mathbf{h} = \text{MaxPool}(\text{GNN}(\mathbf{X}, \mathbf{A}))$。  
- 权重更新：$\mathbf{W} = \hat{\mathbf{W}} + (\mathbf{L} + \mathbf{G} \odot \mathbf{M})\mathbf{R}^\top$，其中 $\mathbf{G} = g(\mathbf{h}; \Theta)$。  
- 范数约束避免 $\mathbf{G}$ 在训练中无限增长，维持原始模型知识。

### 6. 主要结论与发现

- 权重级注入范式有效补充了输入级注入丢失的全图信息，在零样本迁移任务上一致超越所有基线。  
- 生成的权重更新具备数据集特异性（图 A8），表明成功编码了图结构特征。  
- 理论揭示并实验验证了权重生成中的方向保持范数放大问题，所提出的范数重标定简单有效，缓解了过拟合，显著提升了多训练数据下的迁移性能。  
- 权重注入位置在中间层（第 21 层）效果最优；左矩阵生成优于右矩阵生成；全图特征作输入比子图特征更具互补性。  
- 计算开销可控，与纯输入注入方法相比时间增加微弱。
